## üìã Description

A responsive portfolio website built with Flutter showcasing my projects, skills, and contact information. Designed for both web and mobile, with smooth navigation, custom UI components, and adaptive layouts.

### Features

- Responsive design for desktop and mobile
- Custom UI built with Flutter
- Easy navigation using GetX routing
- External links opened using `url_launcher`
- Beautiful fonts via Google Fonts
- Fast, adaptive layout
- Organized project sections

---

## üõ† Technologies

- **Flutter**: Main UI framework
- **GetX**: Routing and state management
- **HTTP**: For network calls
- **Google Fonts**: Custom font styling
- **url_launcher**: Opening URLs externally

## üöÄ Deploy, Run & Clone

#### 1. **Clone the repository**

```bash
git clone https://github.com/md-ruhulamin/my_portfolio.git
cd my_portfolio
```

#### 2. **Install Flutter**

Make sure you have Flutter installed. See the [Flutter official installation guide](https://flutter.dev/docs/get-started/install).

#### 3. **Install dependencies**

```bash
flutter pub get
```

#### 4. **Run on Web**

```bash
flutter run -d chrome
```

#### 5. **Run on Mobile**

Ensure you have an emulator (Android/iOS) running, or connect a device:

```bash
flutter run
```
